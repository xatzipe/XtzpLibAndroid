# Library Overview

XtzpLibAndroid provides the following main components:

## SimpleCalculator
A simple utility class for basic arithmetic operations.

### Example
```kotlin
val calc = SimpleCalculator()
val sum = calc.add(1, 2)
val product = calc.multiply(3, 4)
```
